# License Policy

## Current Decision

This repository is private/all-rights-reserved by default.

The active license file is `LICENSE`. No open-source license is granted for code, documentation, corpus data, schemas, generated artifacts, or governance materials unless a later approved license file or signed agreement explicitly changes that decision.

## Scope

This policy covers:

- source corpus files under `QuizBank/`;
- governance and product documentation;
- local tooling and tests;
- generated manifests, reports, schemas and API seed artifacts.

## Change Control

Changing this policy to an open-source or shared-source license requires an explicit repository-owner decision and review of:

- corpus ownership and reuse rights;
- documentation and generated-artifact publication rights;
- third-party dependency/license compatibility;
- privacy, security and commercial constraints.

Until that review is complete, external distribution, sublicensing, public reuse and derivative publication are not granted.
